🧮 AMC Diagram Renderers

Structured SVG renders — no AI images. Each diagram type maps to a media_spec in the content pack JSON.

1. Symmetry / Letter Shapes (Q5 style)

media_spec: { type: "symmetry_letters", letters: ["V","W","X","Y","Z"], show_lines: true }

2. Circle Arrangement (Q9 style)

media_spec: { type: "circle_arrangement", count: 10, labels: ["1".."10"], highlight: 9 }

3. Folded Paper (Q12 style)

media_spec: { type: "folded_paper", folds: 2, cut_position: "corner", result_shape: "diamond" }

4. Tangram / Grid overlay (Q18 style)

media_spec: { type: "grid_overlay", grid: "4x4", shapes: [...tangram_pieces], highlight: "small_square" }

5. Matchstick Cubes (Q29 style)

media_spec: { type: "matchstick_cubes", count: 3, show_dots: true }

6. Connected Circles / Star (Q20 style)

media_spec: { type: "connected_circles", positions: [{x,y}...], edges: [{from,to}...] }

Each render is ~20 lines of SVG. Generator outputs structured media_spec → frontend renders SVG. No AI image generation needed.
